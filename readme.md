## dudu-cli
创建 vue 多页应用

vue multipage

```
$ npm install dudu-cli -g
$ dudu-cli -v
dudu-cli version 0.0.1
```

安装并启动 helloworld 新项目
```
$ dudu-cli init
$ cd helloworld
$ yarn
$ yarn serve
```


多页中使用各常用组件demo
- createjs 使用 createjs 库 

- fullpage 使用 fullpage 库

- index 默认页

- jsx 使用 jsx 语法

- normal 普通页面

- spa 多页中的单页应用

- video 使用腾讯视频播放器

- virtual-scroll 使用虚拟滚动列表
